# SoVa-nik.github.io

## Запуск локального сервера

Для работы формы авторизации требуется Node.js. После установки зависимостей
можно запустить сервер командой:

```bash
npm install
npm start
```

Сервер запустится на `http://localhost:3000` и будет обслуживать статические
файлы сайта, а также эндпоинты `/api/login` и `/api/register`.

При первом запуске будет создан аккаунт администратора с логином `admin` и
паролем `admin`.
